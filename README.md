# Escola-Nova-Era-Tec-desafios
Este projeto é uma página pessoal desenvolvida com HTML semântico, criada como parte do curso de Formação Front-End.
A estrutura inclui header, main, sections e footer, apresentando minha foto, bio e links para redes sociais.
O objetivo é praticar boas práticas de marcação e organização de conteúdo, mantendo o código simples, acessível e bem estruturado.
